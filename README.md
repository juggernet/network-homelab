# Network Homelab

Repositorio personal de laboratorios de redes, seguridad y automatización realizados en un entorno de pruebas utilizando EVE-NG.

## Objetivo

Documentar la implementación y configuración de diferentes tecnologías utilizadas en entornos empresariales, incluyendo:

- Cisco IOS
- Fortinet FortiGate
- Linux (Ubuntu/Xubuntu)
- Ansible
- Zabbix
- VLANs y Routing
- NAT
- DHCP
- Firewall Policies
- Monitoreo de red
- Automatización de infraestructura

## Entorno de laboratorio

- EVE-NG
- Cisco IOL L2/L3
- FortiGate VM
- Ubuntu Server/Desktop
- Zabbix Server (próximamente)

## Laboratorios

| Laboratorio | Estado |
|------------|---------|
| LAN básica con Firewall | Completada |
| LAN Two Tier | Completada |

## Topología Base

Internet → Fortinet → Router → Switch → Hosts Linux/VPCS
